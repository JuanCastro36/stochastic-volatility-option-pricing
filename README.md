# Stochastic Volatility Option Pricing

This project compares the classical Black-Scholes model with a Merton-Garman stochastic volatility model using option and implied volatility data.

The analysis focuses on five large-cap technology stocks:

- AMZN
- TSLA
- GOOGL
- AAPL
- MSFT

The goal is not to build a trading strategy, but to study how stochastic volatility changes the distribution of terminal stock prices compared with the standard Black-Scholes framework.

---

## Project Overview

Black-Scholes assumes constant volatility, which is often too restrictive for real financial markets. In contrast, stochastic volatility models allow volatility to evolve randomly over time.

This project implements a quantitative finance workflow that:

1. Loads option and implied volatility data.
2. Reconstructs the underlying price using put-call parity.
3. Builds a time series of reconstructed spot prices and implied variance.
4. Estimates Merton-Garman stochastic volatility parameters.
5. Computes closed-form Black-Scholes moments.
6. Simulates terminal prices under Black-Scholes.
7. Simulates terminal prices under Merton-Garman stochastic volatility.
8. Compares mean, variance, skewness and kurtosis across models.
9. Visualizes terminal price distributions.

---

## Methodology

### 1. Put-Call Parity Reconstruction

For each stock and date, the underlying price is reconstructed using put-call parity:

\[
S_0 = C - P + K e^{-r \tau}
\]

where:

- \(C\) is the call mid-price,
- \(P\) is the put mid-price,
- \(K\) is the strike price,
- \(r\) is the risk-free rate,
- \(\tau\) is time to maturity.

The reconstructed series is used as an input for the stochastic volatility analysis.

---

### 2. Black-Scholes Benchmark

Under Black-Scholes, the terminal stock price follows a lognormal distribution:

\[
S_T = S_0 \exp\left[\left(r - \frac{1}{2}\sigma^2\right)\tau + \sigma \sqrt{\tau} Z\right]
\]

The project computes both:

- closed-form moments,
- Monte Carlo estimates.

This provides a benchmark against which the stochastic volatility model is compared.

---

### 3. Merton-Garman Stochastic Volatility Model

The Merton-Garman-type model assumes that the variance process evolves randomly:

\[
dV_t = \tilde{\mu} V_t dt + \xi V_t dW_t^Q
\]

and the stock price follows:

\[
dS_t = rS_t dt + \sqrt{V_t}S_t dW_t^R
\]

with correlation:

\[
dW_t^R dW_t^Q = \rho dt
\]

The estimated parameters are:

- \(\tilde{\mu}\): variance drift,
- \(\xi\): volatility of volatility,
- \(\rho\): correlation between price and variance shocks.

---

## Results

The project compares terminal price distributions generated by Black-Scholes and Merton-Garman Monte Carlo simulations.

For most stocks, the two models produce similar short-maturity distributions. However, the stochastic volatility model can generate wider or shifted distributions when estimated volatility dynamics are stronger.

The variance comparison across models shows that the Merton-Garman model may produce higher dispersion than Black-Scholes, especially for more volatile stocks such as TSLA.

## Limitations

This project is educational and analytical.

Some reconstructed price series may show discontinuities caused by option data limitations, contract adjustments, stock splits, or differences between option-implied reconstruction and observed spot prices.

The Merton-Garman parameters are estimated historically from reconstructed price and implied variance series. This is not a full option-price calibration.

The results should not be interpreted as financial advice or as a trading system.
